## ğŸš€ How to Start Development

### 1. Clone Repository

```bash
git clone https://github.com/Burhan-sheikh/kashpages-production.git
cd kashpages-production
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Firebase Setup

1. Create Firebase project: `kashpages-production`
2. Enable Authentication (Email, Google, Phone)
3. Create Firestore database
4. Create Realtime Database
5. Enable Storage
6. Get Firebase config

### 4. Configure Environment

```bash
cp .env.example .env.local
```

Fill in Firebase credentials in `.env.local`

### 5. Deploy Firebase Rules

```bash
firebase login
firebase use --add  # Select your project
firebase deploy --only firestore:rules
firebase deploy --only firestore:indexes
firebase deploy --only database
```

### 6. Start Development Server

```bash
npm run dev
```

Open http://localhost:3000

### 7. Begin Phase 2 Implementation

Start with authentication:
- Create auth context
- Implement sign up page
- Implement login page
- Add Google OAuth
- Test user creation

Refer to `ROADMAP.md` for detailed implementation tasks.
