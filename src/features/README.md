# Features Directory

This directory contains feature-based modules following a domain-driven design approach.

## Structure

Each feature module contains:
- `components/` - Feature-specific React components
- `hooks/` - Custom React hooks for the feature
- `services/` - API calls and business logic
- `types/` - TypeScript interfaces and types
- `utils/` - Helper functions
- `constants/` - Feature-specific constants

## Features

### Authentication (`auth/`)
- Login, signup, password reset
- Google OAuth integration
- Email verification
- Session management

### User Profile (`user/`)
- Profile viewing and editing
- Avatar management
- Account settings
- User pages list

### Page Builder (`page-builder/`)
- Page creation and editing
- Section management
- Design customization
- Draft and publish workflow

### Discovery (`discovery/`)
- Search functionality
- Category browsing
- Location-based discovery
- Filters and sorting

### Admin (`admin/`)
- Dashboard
- Page moderation
- User management
- Analytics
- Platform settings

### Analytics (`analytics/`)
- Page view tracking
- Click tracking
- Engagement metrics
- Reports generation
